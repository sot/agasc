Getting started¶

The most common usage of the agasc package is getting the AGASC catalog data for stars near a location on the sky or by AGASC ID.

The three main functions are:

  • get_agasc_cone(): Get stars within a radius of RA, Dec

  • get_star(): Get information for one star by AGASC ID

  • get_stars(): Get information for a list of stars by AGASC ID
